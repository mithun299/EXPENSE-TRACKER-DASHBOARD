# 📖 PROJECT DESCRIPTION

The **Expense Tracker Dashboard** is a responsive web application developed as part of the **Full Stack Web Development Internship** at **CODTECH IT SOLUTIONS PRIVATE LIMITED**.

The application enables users to efficiently manage their personal finances by recording income and expenses, tracking transaction history, and automatically calculating the available balance. All transaction data is stored securely using **Browser Local Storage**, allowing data to persist without requiring a database.

---

# 🎯 PROJECT OBJECTIVES

* Record income transactions.
* Record expense transactions.
* Calculate available balance automatically.
* Store transaction history.
* Provide an easy-to-use finance management dashboard.
* Demonstrate JavaScript Local Storage functionality.
