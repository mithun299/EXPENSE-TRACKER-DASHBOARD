# 1. INTRODUCTION

The **Expense Tracker Dashboard** is a web-based application developed as part of the **Full Stack Web Development Internship** at **CODTECH IT SOLUTIONS PRIVATE LIMITED**. It helps users monitor, organize, and manage their personal finances through an intuitive and responsive interface.

The application enables users to record income and expenses, calculate the available balance automatically, and store transaction history using **Browser Local Storage**.

---

# 2. OBJECTIVE

The objectives of this project are:

* Record income transactions.
* Record expense transactions.
* Calculate the available balance automatically.
* Store transaction history using Browser Local Storage.
* Develop a responsive and user-friendly finance management application.

# 7. WORKING OF THE APPLICATION

### Step 1

Open the application in any modern web browser.

### Step 2

Enter the transaction details (income or expense).

### Step 3

Click the **Add Transaction** button.

### Step 4

The transaction is saved in Browser Local Storage.

### Step 5

The dashboard automatically updates:

* Total Income
* Total Expenses
* Available Balance

### Step 6

Users can delete transactions whenever required.
